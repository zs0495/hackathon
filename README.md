## 프로젝트 소개
> 복복, 공공복지에 민간복지를 더하다 
> [공공복지혜택모아보기]와 [민간복지혜택모아보기]를 캐릭터와 함께 제공하여 사용자가 손쉽게 자신을 위한 복지 혜택을 찾을 수 있도록 돕는 서비스입니다.  
> 단순히 찾아보는 것뿐만 아니라 필수 서류까지 알려주는 복복을 경험해보고 더 나은 복지생활을 누려보세요.  

<br>

### 👀 주요기능
- **`회원가입`** : 아이디, 비밀번호, 거주지, 생년월일 등 기본 프로필을 필수로 입력 후 나의 키워드와 나의 상황등 추가 프로필은 선택적으로 입력 가능합니다.
- **`메인`** : 모아보기 바로가기 버튼, 복복 이용가이드, FAQ 제공을 통해 복복을 보다 쉽게 이해할 수 있도록 했습니다.
- **`공공복지혜택 모아보기`** : 나의 키워드, 소득분위, 나의 상황을 입력하고 맞춤형 공공복지혜택을 제공합니다.  
- **`민간복지혜택 모아보기`** : 사정상 공공복지에서 누락되는 사람들을 위해 민간복지혜택도 추가하여 복지 사각지대를 해소할 수 있도록 했습니다. 
- **`마이페이지`** : 내가 찜한 복지혜택과 기본프로필, 추가프로필을 수정 할 수 있습니다.
